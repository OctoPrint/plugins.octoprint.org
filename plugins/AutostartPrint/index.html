AutostartPrint

 Autoselect   Backup Scheduler 

Autostart a print job after conneting.

The Plugin starts a print job after the Printer is connected (e.g. after powering up). It selects the newest uploaded file for print.

Support my Efforts

This plugin, as well as my other plugins were developed in my spare time. If you like it, I would be thankful about a cup of coffee :)

For implementation details please visit the homepage.
